# Launch Checklist

## Technical

- [ ] `npm install`
- [ ] `npm run build`
- [ ] `cd functions && npm install && npm run lint:syntax`
- [ ] Firebase project selected
- [ ] `.env.local` configured
- [ ] Firebase Auth Google enabled
- [ ] Firebase Auth Anonymous enabled
- [ ] Firestore created
- [ ] Firestore Rules deployed
- [ ] Firestore Indexes deployed
- [ ] Functions deployed
- [ ] Hosting deployed
- [ ] App Check enabled
- [ ] Domain connected
- [ ] `/privacy`, `/terms`, `/ads` accessible
- [ ] `/replay/:roomCode` accessible

## Gameplay

- [ ] Friendly room create/join
- [ ] Ranked queue with two accounts
- [ ] Move submission through Function
- [ ] Moves subcollection created
- [ ] Forfeit works
- [ ] Timeout works
- [ ] Ranked settlement works
- [ ] Rating only changes once
- [ ] Recent games created
- [ ] Replay page loads moves

## Monetization

- [ ] AdSense application submitted
- [ ] AdSense approved
- [ ] `VITE_ADSENSE_ENABLED=true`
- [ ] Client ID and slot IDs set
- [ ] `ads.txt` updated
- [ ] Ads not placed near board/buttons
- [ ] Invalid traffic monitoring plan
- [ ] CMP configured if required by target regions

## Operations

- [ ] Admin email configured
- [ ] Report review process
- [ ] Abuse ban process
- [ ] Firebase budget alerts
- [ ] Log monitoring
- [ ] Backup/export policy
- [ ] Incident response checklist
